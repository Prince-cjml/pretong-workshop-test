# Pretong Workshop Course Upstream

This repository is the read-only course upstream for the Hybrid ML Rescue workshop.
It is not a student submission repository.

Students should work in their personal assignment repository and run `bash bootstrap.sh` there.
The bootstrap script fetches the versioned course refs from this upstream and creates a `submission` branch from `course/broken-start-v3`.

Public release refs:

- tags: `course/base-v3`, `course/broken-start-v3`, `course/bad-parallel-loader-v3`, `course/recovery-data-v3`, `course/immutable-v3`
- merge branches: `course/native-integration-v3`, `course/training-observability-v3`

The default branch intentionally contains documentation only, so it does not leak the workshop solution.
